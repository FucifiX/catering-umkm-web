# Food Catering Management System

A web-based application built to help local catering MSMEs (UMKM) digitalize their menu and order management.

## 🚀 Features
- **Dynamic Menu Display**: Browse various catering packages.
- **Order Processing**: Integrated order system with total price calculation.
- **Admin Management**: (If applicable) Manage menu items and view incoming orders.
- **Digital Receipt**: Automated order confirmation details.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (SCSS), JavaScript.
- **Backend**: PHP.
- **Database**: MySQL.

## 📁 Project Structure Highlights
- `/assets`: Contains images, CSS, and JS files.
- `/database`: Includes the `.sql` file for database setup.
- `menu_page.php`: The main interface for customers.
- `proses_order.php`: Logic for handling order submissions.
